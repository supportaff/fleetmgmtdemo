Google Sheets Setup

Connect your Google Sheets to store all fleet data

How to get your API Key:
1. Go to console.cloud.google.com
2. Enable Google Sheets API → Create API Key
3. Create a Google Sheet and make it publicly viewable
4. Copy the Sheet ID from its URL
5. Paste below and click Connect
